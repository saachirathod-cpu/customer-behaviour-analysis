# Customer Behaviour & Prediction Analysis

This project uses Python to analyse customer-related data and build a basic predictive model.

## Objective
To explore patterns in data and apply logistic regression for prediction.

## What I did
- Cleaned and processed dataset
- Performed exploratory data analysis (EDA)
- Applied Logistic Regression model
- Interpreted results to understand patterns

## Tools Used
- Python (Pandas, NumPy, Scikit-learn)

## Key Outcome
This project demonstrates how data analysis and basic machine learning can be used to extract insights and make predictions.
